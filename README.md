# FunHive Games Website

A modern, responsive website for FunHive Games - a game development studio specializing in mobile games, web development, and cutting-edge technology solutions.

## Features

- **Modern Design**: Clean, professional design with smooth animations
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Components**: Engaging user interface with hover effects and transitions
- **Contact Form**: Functional contact form with validation
- **Team Showcase**: Beautiful presentation of team members
- **Technology Stack**: Comprehensive display of technologies used
- **Career Section**: Job openings and company benefits
- **Smooth Scrolling**: Seamless navigation between sections

## Technologies Used

- **React.js**: Modern frontend framework
- **CSS3**: Advanced styling with animations and responsive design
- **Font Awesome**: Icons for enhanced visual appeal
- **Google Fonts**: Inter font family for clean typography

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd funhive-games-website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to `http://localhost:3000`

### Building for Production

To create a production build:

```bash
npm run build
```

The build folder will contain the optimized production files.

## Customization

### Company Information

Update the following files to customize company information:

- `src/components/Hero.js`: Company stats and main message
- `src/components/About.js`: Company description and services
- `src/components/Team.js`: Team member information
- `src/components/Contact.js`: Contact details and form
- `src/components/Footer.js`: Footer information

### Styling

- Modify `src/index.css` for global styles
- Update individual component CSS files for section-specific styling
- Change color scheme by updating CSS custom properties

### Contact Form

The contact form is currently set up for frontend display. To make it functional:

1. Set up a backend API endpoint
2. Update the `handleSubmit` function in `Contact.js`
3. Add form validation and error handling
